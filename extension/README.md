# nvim-web Browser Extension

Chrome/Edge extension for quick access to nvim-web.

## Installation

1. Open `chrome://extensions` (or `edge://extensions`)
2. Enable "Developer mode"
3. Click "Load unpacked"
4. Select this `extension` directory

## Features

- Quick popup to open nvim-web
- Create new sessions
- Configurable server URL
- Connection status indicator

## Usage

Click the extension icon to:
- **Open nvim-web**: Opens in a new tab
- **New Session**: Opens with `?session=new`
- **Settings**: Configure your server URL
